# katsulabs-agent-harness-starter

Reusable harness engineering starter for Cursor-based multi-agent development.

## Included

- `.cursor/rules` role templates (`orchestrator`, `contract`, `backend`, `frontend`, `qa`)
- `docs/harness` operating docs (`README`, `agent-hierarchy`, `workflow`, `todo`, `reference-baseline`)
- `.github/PULL_REQUEST_TEMPLATE.md` with hard gates

## Quick Start

1. Copy this repository as your project baseline.
2. Update ticket prefix and branch rule for your team.
3. Replace test gate commands to match your stack.
4. Start with `docs/harness/todo.md` and define active tickets/DoD.
