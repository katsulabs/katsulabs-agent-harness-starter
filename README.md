# katsulabs-agent-harness-starter

Cursor 기반 멀티 에이전트 개발을 위한 재사용 가능한 하네스 엔지니어링 스타터입니다.

## 포함 항목

- `.cursor/rules` 역할 템플릿 (`orchestrator`, `contract`, `backend`, `frontend`, `qa`)
- `.cursor/skills` 검증 스킬 (링크, 규칙 형식, 일관성, PR 게이트)
- `docs/harness` 운영 문서 (`README`, `agent-hierarchy`, `workflow`, `todo`, `getting-started`)
- `.github/PULL_REQUEST_TEMPLATE.md` 하드 게이트 체크리스트

## 빠른 시작

1. 이 저장소를 새 프로젝트의 기본 템플릿으로 복사합니다.
2. 팀 규칙에 맞게 티켓 prefix와 브랜치 규칙을 수정합니다.
3. `docs/harness/todo.md`부터 시작해 활성 티켓과 DoD를 정의합니다.

상세 온보딩 가이드: `docs/harness/getting-started.md`
